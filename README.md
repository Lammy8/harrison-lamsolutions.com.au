# Portfolio Website

## Overview
This project is a **personal portfolio website**, built with **Vite, React, and TypeScript**, and optimized for deployment on **GitHub Pages**.  
The architecture is intentionally frontend-only, with future plans to introduce a backend via **tRPC**.

---

## Key Features

### Architecture
- Pure **static frontend**
- No backend required
- Future-ready for **tRPC integration**

### Custom Domain
- Configured with **GitHub Pages**
- Domain: **lamsolutions.com.au**
- CNAME located at `public/CNAME`
